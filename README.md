# FoodPro

An intelligent, real-time platform to automate, monitor, and optimize food production pipelines with cutting-edge analytics.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Library**: React 19
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
